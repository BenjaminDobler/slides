# Feature Roadmap

## High Priority

### Export
- [ ] PDF export
- [ ] PPTX (PowerPoint) export
- [ ] PNG export of individual slides

### Code Features
- [ ] Syntax highlighting in code blocks (Shiki or Prism)
- [ ] Line highlighting within code blocks (e.g. highlight lines 2-4)

### Presenter Mode Enhancements
- [ ] Speaker notes view (separate window with current slide, next slide, notes, and timer)
- [ ] Presentation timer / clock
- [ ] Overview mode — grid of all slides for quick navigation

### Math / LaTeX
- [ ] KaTeX or MathJax support for math equations (`$E = mc^2$`)

### Slide Directives
- [ ] Per-slide background images/colors (`<!-- bg: url(...) -->`)
- [ ] Per-slide layout classes (centered, cover, section divider, image-left, etc.)
- [ ] Fragment/click animations — reveal bullet points one by one

### Collaboration / Sharing
- [ ] Public share link for a presentation (read-only viewer URL)
- [ ] Embed code / iframe support

## Nice to Have

### Editor
- [ ] Slide reordering via drag-and-drop in thumbnails
- [ ] Duplicate slide button
- [ ] Import from existing markdown files / drag-and-drop `.md` file
- [ ] Dark/light mode toggle in editor
- [ ] Keyboard shortcuts reference overlay

### Presenter
- [ ] Drawing/annotation mode during presentation

## Already Implemented

- [x] Mermaid diagrams
- [x] Two-column layouts (`<!-- columns -->` / `<!-- split -->`)
- [x] Card grid layout (bold-prefixed list items)
- [x] AI-powered slide generation
- [x] AI-powered slide enhancement (rewrite, speaker notes, diagrams)
- [x] Visual AI review and improvement (screenshot-based)
- [x] AI theme generation
- [x] Live preview with Monaco editor
- [x] Slide transition animations (fade, slide, zoom, flip)
- [x] Theme system with custom CSS themes
- [x] Media library with drag-and-drop into editor
- [x] MCP server for AI agent integration
- [x] Gemini / OpenAI / Anthropic provider support
